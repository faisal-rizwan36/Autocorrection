# Autocorrection
### English Auto Correction using C++
A word is taken as input, and 5 autocorrected words are returned.   


## Inputs
1. First: Input T number of test cases.

2. Next line: Enter the word.

## Running 
For running the program
```
g++ autocorrector.cpp
```
then
```
./a.out
```
